# tattwasclock
Tattwas clock. 

Z = meia noite.
A = aurora, nascer do Sol.
N1 = horário que o usuário está vendo o sistema (horário atual), antes de A e depois de Z.
N2 = horário que o usuário est vendo o sistema (horário atual), depois de A e antes de Z. 
P = período de tempo entre A e o horário atual.
