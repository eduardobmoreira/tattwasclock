# tattwasclock
Tattwas clock. 

Z = meia noite.<br>
A = aurora, nascer do Sol.<br>
N1 = horário que o usuário está vendo o sistema (horário atual), antes de A e depois de Z.<br>
N2 = horário que o usuário est vendo o sistema (horário atual), depois de A e antes de Z. <br>
P = período de tempo entre A e o horário atual.<br><br>
